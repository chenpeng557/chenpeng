【行政区划】

　　永州现辖冷水滩、零陵两区和祁阳、东安、双牌、道县、宁远、江永、江华、新田、蓝山9县、金洞、回龙圩2个管理区，188个乡镇，5000多个行政村。 　　

【冷水滩区】冷水滩区辖13个乡镇、6个街道办事处、1个工业园区、1个农业开发区，土地面积1221平方公里，耕地面积31.21千公顷。2009年年末总人口51.87万人，其中城镇人口31.9万人。人口出生率8.7‰，自然增长率6.5‰。年内气温最高38.2℃，最低-1.9℃；年降水量961.9毫米。现为永州市政治、经济、文化中心，素有“湘西南门户”和“潇湘第一城”之称。

　　

【零陵区】零陵区辖12个乡镇，4个街道办事处。土地总面积1959平方公里，耕地面积33.27千公顷。2009年年末总人口为61.16万人，其中：城镇人口26.85万人，城镇化水平达到43.0%。人口出生率10.82‰，人口自然增长率6.49‰。年内最高气温37.9℃，最低气温－1.1℃，年降水量1097.5毫米。

　　

【祁阳县】祁阳县辖28个乡镇（含金洞管理区）。土地总面积2538平方公里，耕地面积43.82千公顷。2009年年末总人口102.6万人，其中城镇人口38.23万人。人口出生率9.65‰，自然增长率4.76‰。年内最高气温39.4℃，最低气温-1.0℃；年降水量1014.1毫米。

　　　

【东安县】东安县辖16个乡镇，1个经济开发区，1个国家森林公园，1个国有林场，土地总面积2219平方公里，其中耕地面积34.17千公顷。2009年年末人口总数61.16万人，其中城镇人口7.60万人，人口出生率为9.7‰，自然增长率为5.2‰。年内气温最高38.5℃，最低-1.8℃，年降水量958.9毫米。

　　　

【双牌县】双牌县辖3个镇9个乡（其中一个瑶族自治乡）2个国有林场和1个国家级森林公园。土地总面积1751平方公里，其中耕地面积7660公顷。2009年年末总人口18.07万人，其中城镇人口3.3万人，农村人口14.77万人。人口出生率12.35‰,人口自然增长率为7‰。年内气温最高38.3℃，最低-2.4℃，年降水量1273.4毫米。

　　　

【道县】道县辖9镇14乡和3个国有农林场，土地面积2442平方公里，其中耕地面积63.38万亩。2009年年末总人口69.48万，其中城镇人口24.39万人。全县人口出生率11.26‰，自然增长率0.69‰。平均气温19.5℃，年降水量1312.5毫米，日照时数1552.5小时。

　　　

【江永县】江永县辖6个镇5个乡（其中4个为瑶族乡），1个国有农场，1个国有林场，1个省级自然保护区和1个国家级森林公园。土地总面积1540平方公里（不含回龙圩，下同），耕地面积24.54万亩。2009年年末人口267991人（少数民族人口169370人，占总人口的63.2%），其中城镇人口84524人，占总人口的31.5%。人口出生率18.2‰，自然增长率6.23‰。年内最高气温37.7℃，最低-4.8℃;年降水量1159.4毫米。

　　　

【江华瑶族自治县】江华瑶族自治县辖22个乡镇，1个国有林场，512个村（居）委会，4个社区。总面积3248平方公里，其中山区面积258千公顷，耕地面积22千公顷。2009年年末总人口47.99万，其中农业人口40.99万，少数民族人口35.03万（其中瑶族人口28.8万）。人口出生率15.18‰，人口自然增长率为10.21‰。年内气温最高37.7℃，最低-1.4℃。年降水量1021.8毫米。

　　　

【宁远县】宁远县辖17个乡镇,其中瑶族乡4个，4个国有林场，687个行政村。土地总面积2510平方公里，耕地面积40.74千公顷。2009年年末全县总人口80.45万人(其中少数民族人口2.98万人)，城镇人口23.27万人,人口出生率8.7‰，自然增长率3.6‰。年内气温最高39℃，最低-2.4℃,年降水量1152.1毫米。

　　　

【新田县】新田县辖12个乡、7个镇、2个国有林场、379个行政村（居委会）。土地总面积1022.4平方公里，耕地面积14.14千公顷。2009年年末总人口41.25万人，人口出生率12.62‰，自然增长率5.69‰。年内气温最高39.1℃，最低-3.0℃；年降水量1239.1毫米。

　　　

【蓝山县】蓝山县辖9个乡（其中少数民族乡6个）、6个镇和5个国有农林茶场。全县土地总面积1806平方公里，耕地面积23.06千公顷。2009年年末总人口36.06万人，其中城镇人口13.7万人，乡村人口22.36万人，城市化水平35.23%，人口出生率10.3‰，死亡率4.9‰，自然增长率5.4‰。年内气温最高37.4℃，最低-2.7℃；年降水量1233.3毫米。

　　　

【金洞管理区】金洞管理区位于湖南省永州市北部，现辖6乡1镇，6个分场2个工区，73个行政村，总人口8.6万，总面积635平方公里。金洞管理区前身是国营永州市金洞林场，自1954年建场以来，先后获得“全国国有林场先进单位”、“全国国有林场百佳单位”、“全国国有林场十大标兵单位”、“全国四大示范林场”等荣誉称号。2006年成立金洞管理区后，经济社会迅猛发展，林区面貌日新月异，人民生活水平不断提高。

　　

　【回龙圩管理区】回龙圩管理区位于湖南永州市南部，介于北纬25°1′—25°8′和东经111°9′—111°11′之间，北与江永县相连，南与广西贺州市接壤。2001年撤销湖南省国营回龙圩农场，正式设立回龙圩管理区，比照县级政府行使职能职权，全面负责管理区域内的行政、经济和社会事务。管理区下辖1个建制镇、6个办事处、1个国有林场。随着洛湛铁路、道贺、衡昆高速公路的修建，管理区将成为客商投资兴业的“金三角”。
